R é a linguagem de programação que usaremos nesta disciplina. Vamos utilizar a linguagem R com a interface do programa RStudio. Primeiro, você deve baixar e instalar a versão do R mais recente para o seu sistema operacional em https://cran.rstudio.com/.
RStudio é um Ambiente Integrado de Desenvolvimento, ou IDE (integrated development environment), para programação em R e outras linguagens. O programa contém um editor e um console para execução de código, além de outras ferramentas para visualização de gráficos e desenvolvimento de código. Depois de instalar o R, faça o download do RStudio Desktop (Open Source License) em https://www.rstudio.com/.
